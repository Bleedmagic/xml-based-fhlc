# Web-Based Guardian and Student Portal with Learning Progress and Communication Tools

A web-based portal for guardians and students to track learning progress and communicate with educators.

## Tech Used

- XML/XSL/DTD/XSD
- HTML
- CSS
- JavaScript
- Bootstrap

## Features

- XML-based
- Register/Login
